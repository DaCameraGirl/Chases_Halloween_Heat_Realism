# Chase Photo Refs

This folder keeps the best in-repo modeling references for the first real `chase.glb`.

## Current Reference Set

- `IMG_1779.jpeg`: front-ish beach selfie, useful for jaw width, smile shape, and overall proportions
- `IMG_1788.jpeg`: front three-quarter beach selfie, useful for nose bridge, mouth width, and face taper
- `IMG_1912.jpeg`: outdoor portrait in dark hoodie, useful for hoodie silhouette, hair mass, and slim build
- `IMG_3266.PNG`: older hoodie photo, useful for tall curl volume and the longer, narrower front silhouette
- `IMG_5634.PNG`: straight-on social screenshot, useful for hairline height, brow placement, and neutral expression

## What To Prioritize

For the first real model, these matter more than tiny detail:

- tall rounded curls
- narrower lower face
- defined jaw taper
- medium-light brown skin tone
- straight nose bridge with a fuller tip
- dark zip hoodie over a dark shirt

## What Not To Overfit

- phone lens distortion
- temporary facial expression
- lighting color shifts between beach and indoor photos
- tiny skin details before the head silhouette is right
