# Chase Head Model Brief

This is the concrete brief for the first real head-first `chase.glb` pass.

## Goal

Get the likeness to read from gameplay distance and medium close-up before worrying about full realism.

The first successful version does **not** need pores, perfect ears, or fully finished clothing folds. It does need the face, hair silhouette, and hoodie read to feel like Chase immediately.

## Likeness Priorities

### Head Shape

- face is longer than it is wide
- lower face narrows instead of staying boxy
- jaw is present but not heavy
- chin is not broad

### Hair

- curls rise up above the crown instead of hugging the skull
- top silhouette is rounded and tall
- hairline stays fairly open on the forehead
- sides are full but not huge compared with the top

### Brow / Eyes

- brows sit fairly straight with a mild downward angle outward
- eye area should feel set into the face, not pasted onto a flat front plane
- avoid oversized cartoon eyes

### Nose

- bridge is relatively straight
- tip is fuller and softer than the bridge
- nose should project forward enough to break the flat silhouette

### Mouth

- mouth width is moderate, not tiny
- upper lip should have real shape
- front view should read as an actual mouth, not a single dark slit

### Neck / Hoodie

- slim neck
- hoodie collar and zipper create a long vertical center line
- shoulders stay slim rather than broad and heroic

## Build Order

1. Head base mesh
2. Jaw and cheek planes
3. Nose block-in
4. Eye socket depth
5. Mouth and lip forms
6. Hair mass
7. Hoodie neckline and torso silhouette
8. Export test `chase.glb`
9. Tune `chase.config.json`

## Good Enough For First Export

Ship the first real GLB once these are true:

- front view reads as Chase more than the placeholder does
- profile does not collapse into a flat face
- curls have believable height and mass
- hoodie silhouette feels close
- the model sits correctly in the runtime after config tuning

## Stop Doing This

Do not spend hours refining:

- fingers
- shoes
- fabric microfolds
- skin pores
- tiny beard noise

If the head silhouette is still wrong, those details are wasted effort.
